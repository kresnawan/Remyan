# Remyan

## A glimpse about the game
A 500 Rummy-like game with rural Javanese (Indonesia) house rules. Played by whether 4 or 3 players, where each player gets 6 (six) cards if played by 4 players or 7 cards if played by 3 players. Javanese people tend to call it just "Remian" or "Jenggesan" ("Remi" is Indonesian absorption for "Rummy").

## Tech-stack
This game is almost 100% written in Rust, with `macroquad` as the client's UI engine, and `axum` for the networking.
